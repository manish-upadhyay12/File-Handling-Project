Python File Handling Project

A command-line Python application that performs CRUD (Create, Read, Update, Delete) operations on files using Python and the pathlib module.

✨ Features
✅ Create a new file
✅ Read file content
✅ Rename existing files
✅ Append data to a file
✅ Overwrite existing file content
✅ Delete files
✅ Handle errors using Exception Handling

🚀 How to Run

Clone the repository:

git clone https://github.com/manish-upadhyay12/File-Handling-Project.git

Open the project folder:

cd File-Handling-Project

Run the program:

python main.py
